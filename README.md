## What is ForgeAI?

ForgeAI transforms Claude Code from a coding assistant into a **complete business operating system**.

One command installs 150+ battle-tested AI skills that handle everything: SEO audits, video marketing, financial reports, legal contracts, lead generation, content creation, and more.

**Stop prompting. Start forging.**

```bash
# Install ForgeAI
npx forgeai install

# Run any skill
forge seo-audit mywebsite.com
forge video "5 tips for coffee shops" --style hormozi
forge invoice --client "Acme Corp" --amount 5000
forge competitor-spy competitor.com
forge contract-review ./contract.pdf
```

## Quick Start

### Install (30 seconds)

```bash
# Option 1: npx (no install needed)
npx forgeai install

# Option 2: Global install
npm install -g forgeai
forge setup

# Option 3: Python
pip install forgeai
forge setup
```

### First Skill (60 seconds)

```bash
# See all available skills
forge list

# Run your first skill
forge seo-audit example.com

# Get help on any skill
forge help video
```

### That's it. You're forging.
